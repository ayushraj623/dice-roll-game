# 🎲 Dice Roll Game

A simple, interactive dice-rolling game built with Python. 

## 🚀 Features
* Simulates rolling a six-sided die using Python's `random` module.
* Modular code design separating core logic from user interaction.
* Lightweight and runs directly in the terminal.

## 📋 Prerequisites
Ensure you have Python installed on your system. You can check your version by running:
`python --version` (Requires Python 3.x)
